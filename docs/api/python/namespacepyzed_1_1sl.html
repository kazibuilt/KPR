Welcome to the ZED SDK API reference documentation!

The ZED SDK is a powerful tool for developing applications that utilize stereo cameras for a wide range of tasks, such as depth sensing, object detection, positional tracking and more.

The SDK is built around the Camera class, which provides access to the camera's video and depth data, as well as APIs for controlling camera settings and capturing frames.

Modules

In addition to the Camera class, the ZED SDK includes several modules that provide functionality for specific features. These modules are designed to be modular and easy to use, allowing developers to quickly integrate the ZED camera into their applications and start building advanced computer vision systems:

  • Video
  • Depth Sensing
  • Core
  • Spatial Mapping
  • Positional Tracking
  • Object Detection
  • Body Tracking
  • Sensors
  • Fusion
